SMART GAMES Ltda.

Desenvolvedor: Andrey Alves.

==========================================

Projeto desenvolvido para a conclusão do processo seletivo.

Portal da loja Smart Games, uma empresa vendedora de games de diversas plataformas.

-> Nesse portal é necessário a entrada com um login para conseguir o seu acesso
(Informações para login estão ao fim do README);

-> Tendo acesso ao portal é possivel:
	- visualizar jogos no inicio ou pelo menu através de suas categorias
	ou plataformas;

	- Realizar pesquisas de qualquer jogo (cadastrado) por nome, preço ou
	plataforma;
	
	- Ver detalhes do jogo selecionado com informações como: imagem, nome, 
	descritivo e preço;

	- Obter a localização das lojas onde está disponivel a venda daquele jogo
	através de um mapa para a melhor visualização.

-> *OBSERVAÇÕES*: 
	- Cadastro de usuário possui apenas layout e não funcionalidade pois não foi
	pedido pelo cliente;  
	
	- Projeto Mobile não pode ser concluido devido ao curto prazo que foi
	estabelecido pelo cliente;
